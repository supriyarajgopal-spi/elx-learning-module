Views filter views
----------------------

This module allows in views to filter out/by the result of another views.

-- Requirements --

* Views

-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.

-- Configuration --

* If you add a view you can add the views: view filter.
* On the configuration of this filter you can choose which view you want to use
* Additional you can select whether you get only items in the result or exclude items from the result.

That's it.
